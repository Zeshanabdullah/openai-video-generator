## Tutorial

To get started with the OpenAI Video Generator, follow these simple steps:

1. **Installation**:
   To install the OpenAI Video Generator, you can use pip:
   
   pip install openai-video-generator
   

2. **API Key Setup**:
   Make sure you have your OpenAI API key. You can set it as an environment variable:
   bash
   export OPENAI_API_KEY='your_api_key_here'
   

3. **Basic Usage**:
   Here is a basic example of how to generate a video:
   python
   from openai_video_generator import VideoGenerator

   generator = VideoGenerator()
   video = generator.create_video(prompt="A sunset over the mountains", duration=10)
   video.save("sunset_video.mp4")
   

4. **Advanced Options**:
   You can customize the generated video by specifying parameters like `resolution`, `frame_rate`, and `style`:
   python
   video = generator.create_video(
       prompt="A busy city street at night",
       duration=15,
       resolution="1920x1080",
       frame_rate=30,
       style="cinematic"
   )
   

5. **Running the Script**:
   Save your script and run it in your terminal to generate the video.
